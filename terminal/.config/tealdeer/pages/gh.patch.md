- dotfiles: Nutzt `fzf (Interactive), bat (Diff-Highlighting), fzf-Helper (gh)`

- dotfiles: Repository im Browser öffnen:

`gh-open`

- dotfiles: GitHub Status – zugewiesene Issues, PRs, Mentions:

`gh-status`

- dotfiles: PRs durchsuchen (`<Enter>` Checkout/Browser, `<Ctrl d>` Diff, `<Ctrl o>` Browser, `<Ctrl s>` Open↔Alle):

`gh-pr {{suche}}`

- dotfiles: Issues durchsuchen (`<Enter>` Browser, `<Ctrl e>` Bearbeiten, `<Ctrl s>` Open↔Alle):

`gh-issue {{suche}}`

- dotfiles: Actions Runs (`<Enter>` Logs, `<Ctrl r>` Rerun, `<Ctrl o>` Browser):

`gh-run {{suche}}`

- dotfiles: Repo Browser (`<Enter>` Klonen, `<Ctrl o>` Browser):

`gh-repo {{suche}}`

- dotfiles: Gists durchsuchen (`<Enter>` Anzeigen, `<Ctrl e>` Bearbeiten, `<Ctrl o>` Browser):

`gh-gist {{suche}}`
