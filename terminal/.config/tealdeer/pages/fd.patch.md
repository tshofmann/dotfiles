# dotfiles: Aliase

- dotfiles: Nur Dateien suchen:

`fdf {{muster}}`

- dotfiles: Nur Verzeichnisse suchen:

`fdd {{muster}}`

- dotfiles: Inklusive versteckter Dateien:

`fdh {{muster}}`

- dotfiles: Uneingeschränkt (alle Dateien inkl. .gitignore):

`fda {{muster}}`

- dotfiles: Shell-Skripte finden:

`fdsh`

- dotfiles: Python-Dateien finden:

`fdpy`

- dotfiles: JavaScript/TypeScript-Dateien finden:

`fdjs`

- dotfiles: Markdown-Dateien finden:

`fdmd`

- dotfiles: JSON-Dateien finden:

`fdjson`

- dotfiles: YAML-Dateien finden:

`fdyaml`

# dotfiles: Funktionen

- dotfiles: Verzeichnis interaktiv wechseln (`<Enter>` Wechseln):

`cdf {{pfad}}`

- dotfiles: Datei mit Standard-App öffnen (`<Enter>` Öffnen):

`fo {{pfad}}`
