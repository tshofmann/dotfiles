# dotfiles: Globale Tastenkürzel (in allen fzf-Dialogen)

- dotfiles: Vorschau ein-/ausblenden:

`<Ctrl />`

- dotfiles: Alle Einträge auswählen (Multi-Select):

`<Ctrl a>`

- dotfiles: Einzelnen Eintrag zur Auswahl hinzufügen:

`<Tab>`

# dotfiles: Funktionen (aus fzf.alias)

- dotfiles: Aliase und Funktionen durchsuchen (`<Enter>` Definition, `<Ctrl y>` Kopieren, `<Ctrl t>` tldr):

`fa {{suche}}`

- dotfiles: Verzeichniswechsel mit zoxide-Daten (`<Enter>` Wechseln, `<Ctrl d>` Löschen, `<Ctrl y>` Kopieren):

`zf`

- dotfiles: Prozesse beenden (`<Enter>` Beenden, `<Tab>` Mehrfach, `<Ctrl k>` SIGKILL):

`fkill`

- dotfiles: Man-Pages durchsuchen (`<Enter>` Öffnen, `<Ctrl l>` tldr, `<Ctrl b>` man):

`fman`

- dotfiles: Umgebungsvariablen durchsuchen (`<Enter>` Kopieren, `<Ctrl y>` Nur Wert):

`fenv`

- dotfiles: Historie durchsuchen, ins Edit-Buffer laden (`<Enter>` Ins Edit-Buffer, `<Ctrl y>` Kopieren):

`fhist`

# dotfiles: Tool-spezifische fzf-Funktionen

- dotfiles: Siehe `tldr fd` für `cdf`, `fo`
- dotfiles: Siehe `tldr rg` für `rgf`
- dotfiles: Siehe `tldr git` für `glog`, `gbr`, `gst`, `gstash`
- dotfiles: Siehe `tldr gh` für `ghpr`, `ghis`, `ghrun`, `ghrepo`, `ghgist`
- dotfiles: Siehe `tldr brew` für `bip`, `bup`, `brp`, `bsp`
