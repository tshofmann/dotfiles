- dotfiles: Nutzt `fzf (7zf), fd (7zf)`

- dotfiles: Archiv erstellen (7za archiv.7z datei1 datei2):

`7za`

- dotfiles: Archiv entpacken (mit Verzeichnisstruktur):

`7zx`

- dotfiles: Archiv-Inhalt auflisten:

`7zl`

- dotfiles: Archiv testen (Integrität prüfen):

`7zt`

- dotfiles: RAR-Archiv entpacken (macOS kann kein RAR nativ):

`unrar`

- dotfiles: Archiv Browser (`<Enter>` Entpacken, `<Ctrl y>` Pfad kopieren):

`7zf {{pfad}}`
