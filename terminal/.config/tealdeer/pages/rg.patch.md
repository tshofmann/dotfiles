# dotfiles: Aliase

- dotfiles: Suche mit 3 Zeilen Kontext:

`rgc {{muster}}`

- dotfiles: Case-insensitive Suche:

`rgi {{muster}}`

- dotfiles: Suche in allen Dateien ohne Einschränkungen:

`rga {{muster}}`

- dotfiles: Inklusive versteckter Dateien:

`rgh {{muster}}`

- dotfiles: Nur Dateinamen mit Treffern:

`rgl {{muster}}`

- dotfiles: Treffer pro Datei zählen:

`rgn {{muster}}`

- dotfiles: In TypeScript/JavaScript:

`rgts {{muster}}`

- dotfiles: In Python-Dateien:

`rgpy {{muster}}`

- dotfiles: In Markdown-Dateien:

`rgmd {{muster}}`

- dotfiles: In Shell-Skripten:

`rgsh {{muster}}`

- dotfiles: In Ruby-Dateien:

`rgrb {{muster}}`

- dotfiles: In Go-Dateien:

`rggo {{muster}}`

# dotfiles: Funktionen

- dotfiles: Live-Grep mit fzf (`<Enter>` Datei öffnen):

`rgf {{initiale-suche}}`
