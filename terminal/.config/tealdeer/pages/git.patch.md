- dotfiles: Nutzt `fzf (Interactive), bat (Diff-Highlighting)`

- dotfiles: Dateien zum Staging hinzufügen:

`git-add`

- dotfiles: Einen neuen Commit erstellen:

`git-commit`

- dotfiles: Commit mit Nachricht:

`git-cm`

- dotfiles: Alle Änderungen stagen und einen Commit erstellen:

`git-acm`

- dotfiles: Letzten Commit ergänzen (staged Dateien hinzufügen, Message beibehalten):

`git-amend`

- dotfiles: Änderungen pushen:

`git-push`

- dotfiles: Änderungen pullen:

`git-pull`

- dotfiles: Branch wechseln oder Datei zurücksetzen:

`git-co`

- dotfiles: Status des Repositories anzeigen:

`git-status`

- dotfiles: Änderungen anzeigen:

`git-diff`

- dotfiles: Commit-History (`<Enter>` Anzeigen, `<Ctrl y>` SHA kopieren):

`git-log {{suche}}`

- dotfiles: Branch wechseln (`<Enter>` Checkout, `<Ctrl d>` Löschen):

`git-branch {{suche}}`

- dotfiles: Status (`<Enter>` Add, `<Tab>` Mehrfach, `<Ctrl r>` Reset):

`git-stage {{suche}}`

- dotfiles: Stash-Browser (`<Enter>` Apply, `<Ctrl p>` Pop, `<Ctrl d>` Drop):

`git-stash {{suche}}`
