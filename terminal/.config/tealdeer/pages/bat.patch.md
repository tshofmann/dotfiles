# dotfiles: Aliase (bat mit Syntax-Highlighting)

- dotfiles: Dateiinhalt anzeigen (Alias für `bat -pp`):

`cat {{pfad/zur/datei}}`

- dotfiles: Mit Zeilennummern anzeigen:

`catn {{pfad/zur/datei}}`

- dotfiles: Mit Git-Diff-Markierungen:

`catd {{pfad/zur/datei}}`

- dotfiles: Theme interaktiv auswählen und aktivieren:

`bat-theme`
